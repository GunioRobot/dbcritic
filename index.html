DB Critic by Mark Gardner

Critique a database schema for best practices

Dependencies

  • perl 5.10.1
  • Modern::Perl
  • Moo
  • DBIx::Class
  • DBIx::Class::Schema::Loader
  • Getopt::Long::Descriptive
  • Const::Fast
  • List::MoreUtils
  • Scalar::Util
  • Module::Pluggable
  • Algorithm::Combinatorics
  • Data::Compare
  • Devel::Symdump
  • namespace::autoclean

Install

cpanm DBIx::Class::Schema::Critic
